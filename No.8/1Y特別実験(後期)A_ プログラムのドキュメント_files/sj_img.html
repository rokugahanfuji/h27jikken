■■ 画像処理■■

■準備

(1) 必要なファイルを下記よりダウンロードする。
imgpro.tar.gz
kterm を起動し，プログラムを展開し，コンパイルする。
ec50% zcat imgpro.tar.gz | tar xvf -
ec50% cd imgpro
ec50% make

■画像ファイル

用意される画像ファイルは以下の６つ。
kanji.pgm
image1s.pgm
image2s.pgm
image3s.pgm
fruits.pgm
czp.pgm

■コマンド使用法

◆表示
◆濃度変換
　　ヒストグラム画像作成・ ２値化・ 濃度値ｎ倍
　　濃度値線形変換・ ヒストグラム平坦化・ 濃度値反転
◆フィルタリング
　　１次微分・ ２次微分・ テンプレート輪郭抽出
　　平滑化（移動平均フィルタ，メディアンフィルタ）
◆２値処理
　　細線化・ ラベリング・ 特徴検出
　　特徴による抽出
◆マスク処理
◆幾何学変換
　　拡大・縮小（最近傍法，線形補間法）・ 位置の移動・ 回転
　　アフィン変換（拡大縮小・回転・移動）・ 透視変換
◆標本化・量子化
　　再標本化・ 再量子化
◆フーリエ変換
　　ＦＦＴ・ ＦＦＴフィルタ

■ display（画像の表示）

display　画像ファイル

【例】 kanji.pgm を表示
display kanji.pgm

終了するには Ctrl キーを押しながら Q を押す。
または，画面中で左クリックするとメニューが現れるので， File から Quit を選ぶ。

■ histimage（ヒストグラム画像作成）

histimage　[オプション]　入力画像　出力画像

オプション

-s
平滑化を行なう。

【例】kanji.pgm をヒストグラムを平滑化し，kekka.pgm に保存。
histimage -s kanji.pgm kekka.pgm

■ threshold（２値化）

threshold　[オプション]　入力画像　出力画像

オプション

-t n
しきい値を n にする（デフォルト 128）。
-r
白黒反転する。

【例】しきい値 140 で kanji.pgm を反転２値化し，kekka.pgm に保存。
threshold -t 140 -r kanji.pgm kekka.pgm

【例】ラベリング画像 label.pgm からマスク画像を作成し mask.pgm に保存。
threshold -t 99 label.pgm mask.pgm

■ amplify（濃度値n倍）

amplify　[オプション]　入力画像　出力画像

オプション

-n x
濃度値を x 倍する（デフォルト 2）

【例】kanji.pgm の濃度値を 1.5 倍して，kekka.pgm に保存。
amplify -n 1.5 kanji.pgm kekka.pgm

■ liexpand（濃度値線形変換）

liexpand　[オプション]　入力画像　出力画像

オプション

-h high
濃度値 high を最大値（255）に写す（デフォルト 自動検出）
-l low
濃度値 low を最小値（0）に写す（デフォルト 自動検出）

【例】kanji.pgm の濃度値[65,120]を[0,255]に写して，kekka.pgm に保存。
liexpand -h 120 -l 65 kanji.pgm kekka.pgm

■ histplane（ヒストグラム平坦化）

histplane　入力画像　出力画像

【例】kanji.pgm をヒストグラム平坦化して，kekka.pgm に保存。
histplane kanji.pgm kekka.pgm

■ reverce（濃度値反転）

reverce　入力画像　出力画像

【例】kanji.pgm の濃度値を反転し，kekka.pgm に保存。
histimage -s kanji.pgm kekka.pgm

■ gradient（１次微分）

gradient　[オプション]　入力画像　出力画像

オプション

-r
roberts のオペレータを使う。
-s
sobel のオペレータを使う。
-a amp
微分した濃度値を amp 倍する。

【例】kanji.pgm を微分し，見やすいよう濃度値を 4 倍し，kekka.pgm に保存。
gradient -a 4 kanji.pgm kekka.pgm

■ laplacian（２次微分）

laplacian　[オプション]　入力画像　出力画像

オプション

-t n
オペレータ n を使う（1≦n≦3，デフォルト 1）
1 :  0 -1  0    2 : -1 -1 -1   3 :  1 -2  1
    -1  4 -1        -1  8 -1       -2  4 -2
     0 -1  0        -1 -1 -1        1 -2  1
-a amp
微分した濃度値を amp 倍する。

【例】kanji.pgm をオペレータ 2 で微分し，見やすいよう濃度値を 3 倍し，kekka.pgm に保存。
laplacian -t 2 -a 3 kanji.pgm kekka.pgm

■ template（テンプレート輪郭抽出）

template　[オプション]　入力画像　出力画像

オプション

-a amp
処理した濃度値を amp 倍する。

【例】kanji.pgm の輪郭を抽出し，見やすいよう濃度値を 2 倍し，kekka.pgm に保存。
template -a 3 kanji.pgm kekka.pgm

■ smooth（平滑化）

smooth　[オプション]　入力画像　出力画像

オプション

-i
移動平均フィルタを用いる（デフォルトはメディアンフィルタ）。

【例】kanji.pgm を移動平均フィルタを用いて平滑化し，kekka.pgm に保存。
smooth -i kanji.pgm kekka.pgm

■ thinning（細線化）

thinning　入力画像　出力画像

【例】２値画像 nichikanji.pgm を細線化し，kekka.pgm に保存。
thinning nichikanji.pgm kekka.pgm

■ labeling（２値画像のラベリング）

labeling　入力画像　出力画像

【例】２値画像 nichi.pgm をラベリングして，ラベリング画像を kekka.pgm へ保存。
labeling nichi.pgm kekka.pgm

■ features（特徴検出）

features　入力画像

結果

no.
物体の番号
S
面積
L
周囲長
R
円形率
G(x,y)
重心の位置 (x, y)

【例】ラベリング画像 label.pgm の各物体の特徴量を計算する。
features label.pgm

■ features（特徴による抽出）

features　[オプション]　入力画像　出力画像

オプション

-sh n
面積 n 以下のものを抽出。
-sl n
面積 n 以上のものを抽出。
-lh n
周囲長 n 以下のものを抽出。
-ll n
周囲長 n 以上のものを抽出。
-rh n
円形率 n 以下のものを抽出。
-rl n
円形率 n 以上のものを抽出。
-xh n
重心のx座標 n 以下のものを抽出。
-xl n
重心のx座標 n 以上のものを抽出。
-yh n
重心のy座標 n 以下のものを抽出。
-yl n
重心のy座標 n 以上のものを抽出。

【例】ラベリング画像 label.pgm から，面積 100 以上 120 以下のものを抽出し，kekka.pgm に保存。
features -sh 120 -sl 100 label.pgm kekka.pgm

【例】ラベリング画像 label.pgm から，円形度が 0.5 以下のものを抽出し，kekka.pgm に保存。
features -rh 0.5 label.pgm kekka.pgm

■ mask（マスクを使って画像を抜き出す）

mask　[オプション]　入力画像　出力画像

オプション

-f mask_file
マスクに用いる２値画像のファイル名を指定する（デフォルト mask.pgm）

【例】マスク画像 mask.pgm を用いて fruits.pgm から画像を抜き出し，kekka.pgm に保存。
mask fruits.pgm kekka.pgm

■ scale（拡大・縮小）

scale　[オプション]　入力画像　出力画像

オプション

-n
最近傍法を用いる（デフォルト　線形補間法）
-zx x_scale
横方向の拡大率を x_scale にする（デフォルト 1.0）。
-zy y_scale
縦方向の拡大率を y_scale にする（デフォルト 1.0）。

【例】czp.pgm を縦横２分の１に縮小し，kekka.pgm に保存。
scale -zx 0.5 -zy 0.5 czp.pgm kekka.pgm

【例】image2s.pgm を最近傍法を用いて縦４倍，横２倍に拡大し kekka.pgm に保存。
scale -n -zx 2 -zy 4 image2s.pgm kekka.pgm

■ move（位置の移動）

move　[オプション]　入力画像　出力画像

オプション

-px x_move
横方向の移動量を x_move にする（デフォルト 0.0）。
-py y_move
縦方向の移動量を y_move にする（デフォルト 0.0）。

【例】kanji.pgm を横に5，縦に10移動し，kekka.pgm に保存。
move -px 5 -py 10 czp.pgm kekka.pgm

■ rotation（回転）

rotation　[オプション]　入力画像　出力画像

オプション

-r deg
回転角度を deg [°]にする（デフォルト 0°）。

【例】kanji.pgm を15°回転し，kekka.pgm に保存。
rotation -r 15 kanji.pgm kekka.pgm

■ affine（アフィン変換）

affine　[オプション]　入力画像　出力画像

オプション

-px x_move
横方向の移動量を x_move にする（デフォルト 0.0）。
-py y_move
縦方向の移動量を y_move にする（デフォルト 0.0）。
-zx x_scale
横方向の拡大率を x_scale にする（デフォルト 1.0）。
-zy y_scale
縦方向の拡大率を y_scale にする（デフォルト 1.0）。
-r deg
回転角度を deg [°]にする（デフォルト 0°）。

■ perspect（透視変換）

perspect　[オプション]　入力画像　出力画像

オプション

-zx x_scale
横方向の拡大率を x_scale にする（デフォルト 1.0）。
-zy y_scale
縦方向の拡大率を y_scale にする（デフォルト 1.0）。
-px x_move
x方向の移動量を x_move にする（デフォルト 0.0）。
-py y_move
y方向の移動量を y_move にする（デフォルト 0.0）。
-pz z_move
z方向の移動量を z_move にする（デフォルト 0.0）。
-rz z_deg
z軸回りの回転角度を z_deg [°]にする（デフォルト 0°）。
-rx x_deg
x軸回りの回転角度を x_deg [°]にする（デフォルト 0°）。
-ry y_deg
y軸回りの回転角度を y_deg [°]にする（デフォルト 0°）。
-v viewpoint
視点の位置を (0, 0, viewpoint) にする（デフォルト 10）。
-s screen
スクリーンの位置を (0, 0, screen) にする（デフォルト 10）。

【例】kanji.pgm を横方向に 1.5 倍，縦方向に 3.0 倍し，z 軸回りに 10°， x 軸回りに -75°，y 軸回りに 10°回転し，視点を 10，スクリーンを 5 にして， 透視変換し kekka.pgm に保存。
perspect -zx 1.5 -zy 3.0 -rz 10 -rx -17 -ry 10 -s 5 kanji.pgm kekka.pgm

■ sample（再標本化）

sample　[オプション]　入力画像　出力画像

オプション

-h n
横方向のサンプリング間隔を n 倍（整数倍）にする（デフォルト 1）。
-v n
縦方向のサンプリング間隔を n 倍（整数倍）にする（デフォルト 1）。
-f ctl
サンプリングフィルタの制御（0≦ctl≦2, デフォルト 0）。
ctl = 0 : フィルタ無し
ctr = 1 : フィルタあり
ctr = 2 : フィルタのみ（再サンプリング無し）

【例】kanji.pgm を縦横 4 倍の間隔で再サンプリング（フィルタあり）し，kekka.pgm に保存。
sample -h 4 -v 4 -f 1 kanji.pgm kekka.pgm

■ quantize（再量子化）

quantize　[オプション]　入力画像　出力画像

オプション

-b n
n ビットに再量子化にする（1≦n≦8, デフォルト 8）。

【例】kanji.pgm を 4 ビットに再量子化し，kekka.pgm に保存。
quantize -b 4 kanji.pgm kekka.pgm

■ fftimage（ＦＦＴ）

fftimage　入力画像　出力画像


【例】kanji.pgm をＦＦＴし，kekka.pgm に保存。
fftimage kanji.pgm kekka.pgm

■ fftfilter（ＦＦＴフィルタ）

fftfilter　[オプション]　入力画像　出力画像

オプション

-a low
low 以上の周波数を通過する（デフォルト 0）。
-b high
high 以下の周波数を通過する（デフォルト 90.51）。

【例】kanji.pgm にローパスフィルタ（遮断周波数 10）をかけ，kekka.pgm に保存。
fftfilter -b 10 kanji.pgm kekka.pgm

■上級テクニック

上記の各コマンドの出力を省略すると標準出力へ出力します。 さらに入力を省略すると標準入力から入力します。 これを使えば，いくつかの処理を一度に実行することができます。

【例】kanji.pgm をしきい値 148 で２値化して細線化し，kekka.pgm に保存する。
threshold -t 148 kanji.pgm | thinning > kekka.pgm
